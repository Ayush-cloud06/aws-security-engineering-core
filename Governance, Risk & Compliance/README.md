# Governance, Risk & Compliance

This domain defines the governance layer of the security architecture.

Unlike other domains which deploy technical controls, the GRC domain
maps regulatory frameworks and internal policies to the implemented
security controls across the platform.

Frameworks addressed include:

- ISO 27001
- NIST Cybersecurity Framework
- CIS AWS Foundations Benchmark
- GDPR

The documentation in this domain provides:

- Statement of Applicability (SoA)
- Risk management documentation
- Control-to-implementation mappings
- Audit evidence references
