# GDPR Control Mapping (Reference)

## Objective

Map privacy and data protection obligations to platform controls.

| GDPR Principle / Requirement | Platform Control |
|---|---|
| Lawfulness, fairness, transparency | Data governance policy and processing records |
| Data minimization | Access controls and retention rules |
| Accuracy | Data stewardship and correction workflows |
| Storage limitation | Lifecycle and deletion policy |
| Integrity and confidentiality | Encryption, IAM controls, monitoring |
| Accountability | Audit logs, policy evidence, control matrix |

## Supporting Mechanisms

- Data discovery and sensitive data monitoring (Macie)
- Encryption key governance (KMS)
- Auditable access controls and detection telemetry
