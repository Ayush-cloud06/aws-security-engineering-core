# NIST CSF Control Mapping (Reference)

| CSF Function | Platform Capability | Example Control Implementations |
|---|---|---|
| Identify | Asset inventory and risk register | AWS Config inventory, ISMS scope, risk register |
| Protect | Preventive security controls | IAM guardrails, encryption, network controls |
| Detect | Threat and anomaly detection | GuardDuty, Security Hub, CloudWatch detections |
| Respond | Incident response and automation | IR playbooks, EventBridge-driven workflows |
| Recover | Recovery and resilience planning | Backup strategy, recovery testing references |

## Usage

Use this mapping during maturity assessments and control gap analysis.
