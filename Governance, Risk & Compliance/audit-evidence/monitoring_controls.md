# Audit Evidence: Monitoring Controls

## Control Objective

Security monitoring identifies suspicious behavior and supports timely response.

## Evidence Sources

- GuardDuty and Security Hub findings snapshots
- CloudWatch alarms and metric filters
- EventBridge rule configurations for security events
- SOC triage and incident ticket records

## Periodic Checks

- Detection coverage and false-positive trend review
- Alert routing and escalation validation
- Mean time to detect (MTTD) and respond (MTTR) reporting
